# vue_pictures

> A vue.js example project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Start API

Open new terminal tab
Access api/api
Run
```
npm start
```

This API is running


## Other commands
npm install vue-resource@1.0.3 --save
